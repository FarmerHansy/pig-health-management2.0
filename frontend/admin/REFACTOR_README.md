# Admin 后台管理系统 - 简约现代风格重构

## 🎨 设计理念

本次重构采用了**简约现代**的设计风格，以提升用户体验和视觉美感。

### 核心特点

- **极简设计**：减少视觉噪音，专注于核心内容
- **现代配色**：使用清新的蓝色渐变作为品牌色，中性色作为辅助
- **流畅动画**：平滑的过渡效果和交互反馈
- **响应式布局**：适配不同设备和屏幕尺寸
- **统一规范**：全局样式统一体验

## 🎯 主要更新

### 1. 布局系统 (`Layout.vue`)

#### 左侧导航栏
- ✨ **可折叠设计**：支持一键折叠/展开，节省空间
- 🎨 **渐变背景**：从 #2563eb 到 #1e40af 的蓝色渐变
- 📱 **智能展开**：鼠标悬停时自动展开完整菜单
- 💡 **高亮效果**：选中项白色背景 + 蓝色文字，带有阴影反馈

#### 顶部导航栏
- 🎯 **面包屑导航**：清晰展示当前位置层级
- 👤 **用户信息**：头像 + 用户名，简洁美观
- 🔔 **通知入口**：预留通知功能位置（待实现）
- 📐 **折叠按钮**：可切换侧边栏展开状态

#### Logo 设计
- 🐷 **定制图标**：使用 SVG 绘制的猪图标，品牌识别度高
- 📝 **文字标识**：简洁的 "Pig Health" 品牌名称
- 🎭 **动画效果**：折叠时图标保留，文字淡出

### 2. 登录页面 (`Login.vue`)

## 🔧 技术亮点

### 1. 响应式设计
```css
@media (max-width: 768px) {
  /* 移动端适配 */
  .el-table {
    font-size: 13px;
  }
  .el-dialog {
    width: 90%;
  }
}
```

### 2. CSS 变量
使用 CSS 变量统一管理颜色、间距、圆角等，便于维护和主题切换。

### 3. 动画系统
- **过渡动画**：0.3s cubic-bezier(0.4, 0, 0.2, 1)
- **滑入动画**：slideUp, slideIn
- **悬停动画**：transform + shadow

### 4. 可访问性
- 语义化 HTML
- ARIA 标签
- 对比度优化

## 📱 响应式支持

| 设备类型 | 断点 | 适配内容 |
|---------|------|---------|
| 桌面端 | > 768px | 完整功能 |
| 平板 | 768-1024px | 自适应布局 |
| 手机 | < 768px | 简化操作，堆叠布局 |

## 🚀 性能优化

1. **CSS 优化**
   - 使用 CSS 变量，减少重复代码
   - 避免深层嵌套选择器
   - 使用现代 CSS 特性

2. **动画优化**
   - 使用 transform 和 opacity
   - 避免使用 width/height 动画
   - GPU 硬件加速

3. **加载优化**
   - 样式预加载
   - 按需加载图标
   - 代码分割

## 📝 使用指南

### 引入全局样式
在 `main.js` 中已自动引入：
```javascript
import '@/assets/styles/global.css';
```

### 自定义主题色
修改 `global.css` 中的 CSS 变量：
```css
:root {
  --primary: #your-color;
  --primary-dark: #your-dark-color;
  --primary-light: #your-light-color;
}
```

### 新增页面样式
1. 参考已重构的页面结构
2. 使用全局样式类名
3. 保持一致的设计语言
